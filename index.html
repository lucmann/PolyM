class PolyM::Msg

Msg represents a simple message that doesn’t have any payload data. Msg ID identifies the type of the message. Msg ID can be queried with getMsgId().

Subclassed by PolyM::DataMsg< PayloadType >

class PolyM::Queue

Queue is a thread-safe message queue. It supports one-way messaging and request-response pattern.

Public Functions

void put(Msg &&msg)

Put Msg to the end of the queue.

Parameters

msg – Msg to put to the queue.

std::unique_ptr<Msg> get(int timeoutMillis = 0)

Get message from the head of the queue. Blocks until at least one message is available in the queue, or until timeout happens. If get() returns due to timeout, returns a nullptr.

Parameters

timeoutMillis – How many ms to wait for message until timeout happens. 0 = wait indefinitely.

std::unique_ptr<Msg> tryGet()

Get message from the head of the queue. Returns an empty pointer if no message is available.

std::unique_ptr<Msg> request(Msg &&msg, int timeoutMillis = 0)

Make a request. Call will block until response is given with respondTo(). If request() returns due to timeout, returns a nullptr.

Parameters

  • msg – Request message. Is put to the queue so it can be retrieved from it with get().

  • timeoutMillis – How many ms to wait for response until timeout happens. 0 = wait indefinitely.

bool respondTo(MsgUID reqUid, Msg &&responseMsg)

Respond to a request previously made with request(). If the requestID has been found, return true.

Parameters

  • reqUid – Msg UID of the request message.

  • responseMsg – Response message. The requester will receive it as the return value of request().
